# rosrust tutorial

## How to build

```bash
$ cargo build
```

## How to run

### run roscore

```bash
$ roscore
```

### run publisher

```bash
$ cargo run --bin publisher
```

or,

```bash
$ ./target/debug/publisher
```

### run subscriber

```bash
$ cargo run --bin subscriber
```

or

```bash
$ ./target/debug/subscriber
```
